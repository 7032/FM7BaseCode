# functest — 機能デモ集

FM-7 / FM77AV シリーズの各機能を、実機およびエミュレータ上で実際に
動かして確認するための小さなデモ集です。各デモは F-BASIC を使わない
2D ディスクイメージ (`.d77`) として単体で起動し、実機・WebM7・その他の
エミュレータで動作します。

## 収録デモ

| デモ | 内容 |
|------|------|
| [mouse](mouse/) | マウスカーソルデモ。バスマウス / インテリジェントマウスの両対応、画面モード切替ボタン付き。FM-7 / FM77AV / FM77AV40 / FM77AV40EX 対応 |

各デモのフォルダに、操作方法・ビルド方法を記した README、ソース、
起動用ディスクイメージ、および WebM7 の core 上でデモをブートして
自動検証するヘッドレステスト (`make test`) が入っています。
